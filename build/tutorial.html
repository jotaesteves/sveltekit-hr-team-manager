HR Team Manager Tutorial

Learn how to use our comprehensive HR management system

1 Getting Started

Welcome to the HR Team Manager! This system helps you manage all aspects of human resources including employee data, performance reviews, feedback, and one-on-one meetings.

Navigation

Use the sidebar on the left to navigate between different modules. The main modules include People, Reviews, Feedback, One-on-One meetings, and more.

2 Managing People

The People module allows you to manage employee information including:

  • Personal Information: Names, email addresses, positions
  • Employment Details: Department, hire date, active status
  • Search & Filter: Use the search bar to quickly find employees

Pro Tip

Click "Add New Employee" to add team members. Use the checkbox fields to mark employees as active or inactive.

3 Performance Reviews

Manage the complete performance review lifecycle:

  • Review Creation: Set up reviews with employee, reviewer, and period
  • Status Tracking: Monitor review progress from Draft to Completed
  • Scoring: Record overall performance scores and detailed feedback
  • Goal Setting: Document goals and action items for future development

4 Feedback System

Facilitate continuous feedback culture:

  • Multiple Types: Peer, Manager, Upward, and Self-Assessment feedback
  • Anonymous Options: Enable anonymous feedback for honest communication
  • Rating System: Use 1-10 scoring for quantitative assessment
  • Status Management: Track feedback from submission to review

5 One-on-One Meetings

Schedule and track regular one-on-one meetings:

  • Scheduling: Set meeting dates and duration
  • Agenda Planning: Document discussion topics in advance
  • Meeting Notes: Record key discussion points
  • Action Items: Track follow-up tasks and commitments
  • Next Meeting: Schedule recurring meetings efficiently

6 Advanced Features

Performance Grid

Visualize team performance with the interactive performance grid. Compare employee ratings across different criteria with color-coded heatmaps.

Evaluation Criteria

Define custom evaluation criteria with weighted scoring. Set up criteria that align with your organization's values and goals.

Import/Export

Bulk import employee data and export reports for analysis. Track all import/export operations with detailed logs.

Settings Management

Configure system-wide settings including company information, review cycles, and security preferences.

💡 Best Practices

Regular Updates

Keep employee information current and schedule regular review cycles.

Consistent Feedback

Encourage ongoing feedback rather than waiting for formal review periods.

Goal Tracking

Set SMART goals and regularly review progress during one-on-ones.

Data Security

Regularly review access permissions and maintain confidentiality.

Need Help?

Our HR Team Manager is designed to be intuitive, but we're here to help if you need assistance.
